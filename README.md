## Galería Interactiva de Tarjetas

Este proyecto consiste en el desarrollo de una galería interactiva de tarjetas utilizando JavaScript. La aplicación permite crear, visualizar, filtrar y eliminar tarjetas dinámicamente, aplicando conceptos fundamentales de manipulación del DOM y manejo de eventos.

## Tecnologías utilizadas

HTML5
CSS3
JavaScript

## Instrucciones 

Clonar el repositorio
Abrir el archivo index.html en el navegador o usar Live Server en Visual Studio Code
Interactuar con la aplicación desde el formulario

Este proyecto demuestra la capacidad de construir interfaces dinámicas e interactivas utilizando únicamente JavaScript básico, reforzando habilidades esenciales en el desarrollo web.
